# How to use cucumber
## Installation
### Step 1
`npm install`

### Step 2
`Windows search` -> `Systemumgebungsvariablen bearbeiten` -> `Umgebungsvariablen` -> `Path` -> `Neu` -> `C:\path\to\dueto-frontend\tests\`

## Run cucumber tests

### Run all
`npm run cucumber`

### Specific features
`npx cucumber-js features\YOUR_FILE.feature`
`npx cucumber-js features\(.*).feature`
